For adding buttons you must use button shortcode which looks like that: <button>Text</button>.
There is more option with different attributes - style, href, target.

Text: You can change the text of the button. Example: <a href="#">New Text</a>.
Style: You can choose betwen few styles - primary, default, success, info, warning or danger. Example: <a class="default">Text</a>.
Href: You can add any URL to the button. Example: <a href="http://www.google.com">Text</a>.
Target: You can choose if you want to open link in same (_self) or new (_blank) window. Example: <a target="_blank">Text</a>.

Font Awesome Icons

Shortcode for Font Awesome is very simillar to the buttons shortcode. It looks like that: <fa>
There is 3 options with different attributes - icon, href and target.

Icon: You can use any FontAwesome icon. And there is 479 of them. You must use just title of the icon. Example: <fa icon="fa-phone">
Href: You can add URL to the icon. Example: <a href="http://www.proteusthemes.com">
Target: You can choose if you want to open link in the same (_self) or new - (_blank) window. Example: <fa target="_blank">
